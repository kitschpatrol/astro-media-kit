## Video directive

Use `:::Video{...}:::` to embed videos:

:::Video{src="/test.mp4"}
:::

With caption:

:::Video{src="/test.mp4"}
Video **caption** text.
:::

With attributes:

:::Video{src="/test.mp4" controlStyle="minimal" loop=true}
:::

## Audio directive

Use `:::Audio{...}:::` to embed audio:

:::Audio{src="/test.mp3"}
:::

With caption:

:::Audio{src="/test.mp3"}
Audio **caption** text.
:::
